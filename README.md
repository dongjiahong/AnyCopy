# AnyCopy

一个优雅的 macOS 剪贴板历史管理工具。

## 功能特性

- 📋 **剪贴板历史** - 自动记录复制的文字和图片
- 🔍 **快速搜索** - 支持文字内容搜索过滤
- ⌨️ **快捷键唤醒** - 使用 ⇧⌘V 快速打开
- 🖥️ **状态栏模式** - 无 Dock 图标，节省空间
- 🚀 **开机启动** - 系统启动时自动运行
- 💾 **本地存储** - 使用 SQLite 持久化数据

## 系统要求

- macOS 14.0 或更高版本
- Xcode 15.0 或更高版本（用于编译）

## 使用方法

### 编译运行

1. 使用 Xcode 打开 `AnyCopy.xcodeproj`
2. 选择目标设备为 "My Mac"
3. 点击运行按钮 (⌘R)

### 操作说明

- **⇧⌘V** - 全局快捷键唤醒窗口
- 点击状态栏图标打开主窗口
- 在搜索框输入关键词过滤历史记录
- 点击列表项查看详情
- 点击"复制"按钮将内容复制到剪贴板
